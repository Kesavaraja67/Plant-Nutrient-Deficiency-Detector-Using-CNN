# Plant-Nutrient-Deficiency-Detector-Using-CNN
Plant-Nutrient-Deficiency-Detector-Using-CNN A deep learning-based web application built with Streamlit and TensorFlow that detects plant nutrient deficiencies (Nitrogen, Phosphorus, and Potassium) from leaf images. The system uses a CNN model to classify plant health efficiently, helping in early diagnosis and improved crop management.
